🤖 AI Career Guidance Chatbot

🚀 Overview
This project is an **AI-powered Career Guidance Chatbot** built using Jotform AI Agent Builder.  
It helps users discover the most suitable career paths based on their interests, strengths, and goals through an interactive conversational interface.

🎯 Problem Statement
Many students are confused about choosing the right career path due to lack of guidance and clarity.  
This chatbot aims to simplify career decision-making by providing personalized suggestions.

✨ Features
💬 Interactive chatbot interface  
🧠 AI-based personalized career suggestions  
🎯 Interest-based career recommendations  
📚 Required skills for each career  
🤝 User-friendly and beginner-focused guidance  

 ⚙️ How It Works
1. The chatbot asks users about their interests and strengths  
2. It analyzes user responses  
3. Suggests 2–3 suitable career paths  
4. Provides:
   - Required skills  
   - Beginner roadmap  
5. Guides users in choosing the right direction  

🛠️ Tech Stack
- Jotform AI Agent Builder (No-code platform)  
- AI-based conversational logic
